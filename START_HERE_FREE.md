## âš¡ Quick Start (30 Minutes)

### Step 1: Create Accounts (10 min)
1. **Supabase**: [supabase.com](https://supabase.com) - Sign up FREE
2. **Cloudinary**: [cloudinary.com/users/register/free](https://cloudinary.com/users/register/free) - Sign up FREE
3. **Vercel**: [vercel.com/signup](https://vercel.com/signup) - Sign up FREE

### Step 2: Set Up Database (5 min)
1. In Supabase, create new project
2. Open SQL Editor
3. Copy code from `supabase-schema.sql`
4. Paste and run

### Step 3: Configure Project (5 min)
1. Create `.env.local` file
2. Add your Supabase URL and key
3. Add your Cloudinary credentials
4. Save file

### Step 4: Test Locally (5 min)
```bash
npm install
npm run dev
```
Open: http://localhost:3000

### Step 5: Deploy (5 min)
```bash
npm i -g vercel
vercel --prod
```
Add environment variables in Vercel dashboard
